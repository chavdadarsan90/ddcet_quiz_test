# DDCET Quiz & Test Platform

An interactive, secure, and scalable web-based examination system designed specifically for DDCET preparation, built using the Django framework.

## ✨ Features
* **User Authentication:** Secure registration and login for students and examiners.
* **Dynamic Quiz Engine:** Supports multiple-choice questions (MCQs), timers, and instant result generation.
* **Performance Analytics:** Comprehensive dashboards to track quiz history and scores.
* **Admin Control:** Easy-to-use Django admin interface to manage questions, categories, and test schedules.
